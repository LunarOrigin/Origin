# Origin

## La confiance ne se décrète pas. Elle se démontre.

Origin est un projet open source visant à analyser l'origine probable des contenus numériques à l'ère de l'intelligence artificielle.

Le projet a pour objectif de créer des outils transparents et explicables permettant d'analyser des contenus comme les textes, les images et les vidéos.

## Objectifs

- Améliorer la transparence numérique.
- Développer des outils accessibles à tous.
- Encourager la recherche et la collaboration.
- Respecter la vie privée des utilisateurs.

## Vision

Créer une référence ouverte permettant de mieux comprendre l'origine des contenus numériques.
